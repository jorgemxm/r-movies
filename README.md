# R-Movies
Example App using React, ReactRouter & Bulma.css
- **App preview:** <https://r-movies.herokuapp.com/>

---
#### Install:
 - Run `npm i`

---

#### Development
 - Open a terminal tab and type: `npm run dev`
 - Open a secondary tab and run: `npm start`
 - Go to <http://localhost:3000/>

#### Production
 - npm run build-prod
